## Tools for bug bounty

```
This project installs some common bug_bounty tools, to make installing the tools easier.
``` 
## Requirements:

- You need to be the root user
- GNU/Linux
- wget
- git

## How to run

```
chmod +x ./ini.sh; ./ini.sh
```
